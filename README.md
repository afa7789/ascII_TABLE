# ascII_TABLE
a simple c script that will print a ascII table in a nice format.

## To compile on different platforms:

### On macOS:

```gcc -o ascii ascii.c -framework CoreFoundation```

### On Linux:

```gcc -o ascii ascii.c -liconv```

### On Windows:

```gcc -o ascii.exe ascii.c```

## Running it

```
./ascii
./ascii.exe
```
